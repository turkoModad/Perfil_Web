# Perfil Web - José Aníbal Modad

Este proyecto es una **web personal estática** construida con **FastAPI** y HTML, pensada para mostrar mi **currículum vitae**, experiencia laboral, proyectos y redes de contacto.

## Funcionalidades

- ✅ Visualización online de mi CV
- ✅ Enlace para descarga directa del CV en PDF
- ✅ Acceso rápido a mis perfiles:
  - [GitHub](https://github.com/codepyhub)
  - [LinkedIn](https://linkedin.com/in/...)
- ✅ Formulario de contacto con envío de mensajes al administrador vía SMTP

## Tecnologías utilizadas

- **Python 3.11+**
- **FastAPI**
- **Jinja2**
- **Bootstrap 5**
- **Uvicorn**
